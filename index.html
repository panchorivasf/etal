Guide

What etal does

Build a manuscript-ready author list with numbered affiliations. Authors stay in order; affiliation numbers update automatically from first appearance.

Authors

  • Add author — create a blank name field.
  • Drag the handle to reorder; checkboxes select several authors.
  • Sort selected A–Z — alphabetize only the checked authors, keeping their slots.
  • Dedupe — drop repeated names and merge their affiliations.
  • + affiliation — attach an existing affiliation or type a new one.
  • Drag affiliation chips to reorder superscripts for that author.

Affiliations

  • Edit text in the right column. Numbers are assigned by the order authors first use them.
  • Merge identical — combine duplicate affiliation strings.
  • Unused affiliations stay listed but fade; delete when you no longer need them.

Import & export

  • Import DOCX — Word file with an author paragraph (superscript numbers) and affiliation paragraphs below.
  • Export DOCX — same layout, with live Word superscripts.
  • Save / Open project — JSON backup of your full list.

Preview

The dock at the bottom shows the author line and numbered affiliations. Copy author line keeps superscripts when you paste into Word.

Privacy

Everything runs in your browser. Files you open are not uploaded to a server.
